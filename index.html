The reliability loop

Claim. Decide. Gate. Complete. Context compounds across every session and every agent.

1 Task Claimed
$ enginehaus task next
Claimed: Implement auth system
Priority: critical
Context: 3 prior decisions loaded
Agent picks up highest-priority work with full architectural context from prior sessions.
2 Decisions Logged
$ enginehaus decision log \ "JWT with refresh tokens" \ -c architecture
Logged: a3f8c2d1
Linked to: auth system task
Every architectural choice is recorded with rationale. Future agents inherit this reasoning.
3 Context Survives
# New session, hours later $ enginehaus task next
Resumed: auth system
Decisions: JWT + refresh tokens
Progress: Phase 3/8 (Core impl)
Cross-agent memory: session dies, context lives. A different agent, hours later, picks up exactly where the last one stopped.
